# Redis Lite Server
A redis lite server implemented using Ruby threads and Python asyncio as part of the Coding Challenges: https://codingchallenges.fyi/challenges/challenge-redis

## Implemented Commands
- decr
- del
- echo
- exists
- get
- incr
- lpush
- lrange
- ping
- pub/sub
- rpush
- save
- set

Background task to save the file and expire keys.
